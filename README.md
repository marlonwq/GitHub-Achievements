# 🏅 Exibindo Conquistas 🏅

#### Exibir conquistas no seu perfil é completamente opcional; por padrão, elas podem ser vistas por qualquer pessoa que visualizar seu perfil público.

#### Você pode optar por não exibir conquistas no seu perfil acessando suas [configurações de perfil](https://github.com/settings).
